# 🎨 codespaces-html-css

[

![Codespaces Ready](https://img.shields.io/badge/Codespaces-Ready-blue?logo=github)

](https://github.com/features/codespaces)

Entorno HTML & CSS listo para usar en GitHub Codespaces. Sin instalar nada.

## ¿Qué incluye?
- Live Server (previsualización en tiempo real)
- HTML Snippets y CSS IntelliSense
- Auto Close Tag y Auto Rename Tag
- JS básico de complemento
- Formato automático al guardar

## ¿Cómo usarlo?
1. Haz clic en el botón verde **"Code"**
2. Selecciona la pestaña **"Codespaces"**
3. Haz clic en **"Create codespace on main"**
4. Espera a que el contenedor se construya
5. Abre `index.html` y haz clic en **Go Live** en la barra inferior
